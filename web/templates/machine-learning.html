Regresión logística

La regresión logística es un método estadístico utilizado para modelar y predecir la relación entre una variable dependiente categórica y una o más variables independientes continuas o categóricas. A diferencia de la regresión lineal, que se utiliza para predecir valores numéricos continuos, la regresión logística se enfoca en problemas de clasificación binaria o multiclase, donde la variable dependiente toma valores discretos.

Concretamente la regresión logística multiclase es una extensión de la regresión logística binaria que se utiliza para problemas de clasificación con más de dos clases. Este enfoque proporciona una forma efectiva de asignar probabilidades a cada clase en función de las variables independientes como se puede observar en la figura. El modelo estima un conjunto de coeficientes para cada clase y realiza una combinación lineal de las variables independientes ponderadas por estos coeficientes. Además, se debe cumplir que las clases son mutuamente excluyentes y no hay superposición entre ellas ya que, si existiese superposición, se deben considerar otros enfoques de clasificación distintos a este.

Random Forest

El algoritmo Random Forest (Breiman, 2001) es una técnica de aprendizaje supervisado que genera múltiples árboles de decisión sobre un conjunto de datos de entrenamiento: los resultados obtenidos se combinan a fin de obtener un modelo único más robusto en comparación con los resultados de cada árbol por separado.

El proceso de construcción de un Random Forest se divide en dos etapas principales: entrenamiento y predicción. Durante la etapa de entrenamiento, se construyen múltiples árboles de decisión utilizando diferentes muestras aleatorias del conjunto de entrenamiento. En cada nodo del árbol, se selecciona una submuestra aleatoria de características para evaluar la mejor división en función de algún criterio, como la ganancia de información o la reducción del error cuadrático medio. Este proceso se repite recursivamente hasta que se alcance un criterio de parada, como un tamaño mínimo de nodo o una profundidad máxima.

Redes neuronales

Las redes neuronales, también conocidas como redes neuronales artificiales o deep learning, son un enfoque de aprendizaje automático inspirado en el funcionamiento del cerebro humano. Estas redes están compuestas por múltiples capas de neuronas interconectadas, lo que les permite capturar representaciones más complejas y jerárquicas de los datos.

Como se puede observar en la figura, una red neuronal consta de una capa de entrada, una o más capas ocultas y una capa de salida. Cada neurona en cada capa está conectada a todas las neuronas de la capa siguiente mediante conexiones ponderadas. Estas conexiones, también conocidas como pesos, determinan la fuerza y dirección de la influencia de cada neurona en las predicciones realizadas por la red. Además, cada neurona en cada capa puede aplicar una función de activación no lineal, como la función sigmoide o la función ReLU, para introducir no linealidades en el modelo.

KNN

El algoritmo KNN (k-nearest neighbors) es un método de aprendizaje automático supervisado utilizado para clasificación y regresión que tiene objetivo principal clasificar un nuevo punto de datos o predecir un valor continúo basándose en la información proporcionada por los puntos de datos vecinos más cercanos en un espacio de características.

La idea fundamental detrás del algoritmo KNN es que los puntos de datos similares tienden a agruparse en el espacio. Por lo tanto, si conocemos las etiquetas de los puntos de datos vecinos más cercanos a un nuevo punto, podemos inferir su etiqueta o valor basándonos en la similitud con estos vecinos.

K-means

El algoritmo K-means es un método de aprendizaje automático no supervisado utilizado para la agrupación (clustering) de datos. Su objetivo principal es particionar un conjunto de datos en k grupos o clusters, donde cada punto de datos pertenece al clúster más cercano a su centroide.
