ANÁLISIS EXPLORATORIO DE DATOS (EDA)

El siguiente paso en el desarrollo de este proyecto ha sido el análisis exploratorio de datos (Exploratory Data Analysis, EDA). El objetivo de esta etapa es comprender mejor la estructura y características de los datos obtenidos de la primera división de LaLiga española.



Este análisis se ha realizado utilizando el lenguaje de programación Python, aprovechando las bibliotecas y herramientas disponibles que facilitan la manipulación, visualización y análisis de datos.


Cabe destacar que cada archivo descargado de Football-Data correspondiente a una temporada específica contiene información de 380 partidos de la primera división de LaLiga española. Estos archivos se han examinado detalladamente para comprender su estructura y contenido.


Durante el proceso de análisis, se ha prestado especial atención a la estructura de los archivos y se ha observado cómo están organizados y qué variables están presentes en cada uno de ellos. Este paso es esencial para garantizar que los datos de todas las temporadas sean consistentes y puedan ser combinados de manera adecuada.


Además, se han aplicado técnicas de manipulación de datos para combinar los archivos de las diferentes temporadas en un único conjunto de datos coherente. Esto ha permitido tener una visión global de los partidos de la primera división de LaLiga española desde la temporada 2011/2012 hasta la temporada 2021/2022 y realizar un análisis más completo y exhaustivo.


Durante la combinación de los archivos de cada temporada, se ha agregado una nueva variable denominada "Season" que indica a qué temporada pertenece cada partido. Esta adición de información ha sido fundamental para identificar y distinguir los registros correspondientes a cada temporada específica dentro del conjunto de datos consolidado.
